## 프로젝트

**길동무 (Gildongmu)** — 한국인 순례자를 위한 카미노 데 산티아고 여정 관리 서비스.

출국 90일 전 코스 선정부터 현지 도보, 귀국 후 기록까지를 한국어로 관리한다.
경쟁 앱(Buen Camino, Wise Pilgrim)이 "길 위의 도구"에 머무는 사이, **준비 단계**를 장악하는 것이 차별점이다.

**현재 단계: Phase 1 — 웹 일정 계산기.** 앱은 아직 만들지 않는다.

---

## 절대 규칙

### 1. 데이터를 지어내지 않는다 (최우선)

이 서비스는 순례자의 안전과 직결된다. **없는 알베르게를 만들어내거나, 확인되지 않은 거리·요금·전화번호를 채워 넣으면 사람이 한데서 자게 된다.**

- 알베르게 이름, 침대 수, 요금, 전화번호, 개방 기간을 **추정으로 채우지 않는다**
- 데이터가 없으면 `null`로 두고 UI에서 "정보 없음"으로 표시한다
- 샘플/더미 데이터를 넣어야 할 때는 반드시 `source: "PLACEHOLDER"` 로 표시하고, 화면에도 샘플임을 노출한다
- 마을 거리·고도는 `data/towns.ts`에 이미 검증된 값이 있다. **임의로 수정하거나 보간하지 않는다**

### 2. 원어명을 지운다 = 버그

알베르게와 마을 이름은 **스페인어 원어를 반드시 함께 표시**한다. 순례자가 현지에서 화면을 보여줘야 하기 때문이다. 한글만 남기는 UI는 리뷰에서 거절한다.

### 3. 계산 로직은 순수 함수 + 테스트

일정 분할, 거리, 고도 누적은 전부 `lib/planner/` 아래 순수 함수로 두고 단위 테스트를 붙인다. 컴포넌트 안에서 계산하지 않는다.

### 4. 오프라인을 전제로 설계

지금은 웹이지만 Phase 4에서 React Native로 옮긴다. **계산 로직에 네트워크 의존성을 넣지 않는다.** 서버 API 호출로 일정을 계산하는 설계는 금지.

### 5. 접근성 하한선

주 사용자층에 60대 이상이 상당수다.
- 본문 최소 16px, 주요 수치 18px 이상
- 명도 대비 4.5:1 이상
- 터치 타깃 최소 44×44px
- 키보드 포커스 링 제거 금지

## 디자인 토큰

카미노 공식 표지(사아코베오)의 코발트 + 노란 화살표에서 가져왔다.

```
--ink:     #12253F   짙은 코발트. 헤더, 이정표 판
--ink-2:   #1D3B5E
--granite: #E7E8E2   갈리시아 화강암. 페이지 배경
--stone:   #C6C8BE   테두리, 구분선
--flecha:  #F0B429   노란 화살표
--vino:    #7A2E39   도장 잉크
--moss:    #3F5D4A
--text:    #1B2430
--muted:   #6B7280
```

**노란색(`--flecha`) 사용 규칙: 길 안내·현재 위치·활성 상태에만 쓴다.** 실제 카미노에서 노란색은 방향 지시 전용이다. 장식이나 강조에 남용하면 규칙이 무너진다.

**서체**
- 표제: `Gowun Batang` (한글 세리프) — 아껴서 사용
- 본문: `IBM Plex Sans KR`
- 수치: `IBM Plex Mono` — 거리, 고도, 남은 km는 전부 모노스페이스 + `tabular-nums`

**시그니처 요소: 이정표(모호온) 판.** 상단에 산티아고까지 남은 거리를 스페인식 소수점 표기로 표시한다 (`114,235`). 이 컴포넌트는 임의로 바꾸지 않는다.

## 도메인 지식 (틀리기 쉬운 것들)

- **콤포스텔라**: 도보 100km(자전거 200km) 이상이어야 발급. 사리아 출발이 최다 출발지인 이유다
- **크레덴시알(순례자 여권)**: 도장을 모으는 종이 수첩. **사리아 이후 구간은 하루 2개** 필요
- **알베르게 유형**: 공립(municipal) / 본당(parroquial) / 수도원 / 사립 / 기부제(donativo)
- **대부분의 공립 알베르게는 예약을 받지 않는다.** 선착순이므로 "예약 가능"을 잘못 표시하면 치명적
- **겨울 폐쇄 숙소가 많다.** `open_from` / `open_to` 없이 숙소를 표시하면 안 된다
- 프랑스 길 총 773.1km (생장피드포르 → 산티아고). 사리아부터는 114.2km
- 큰 오르막 3곳: 피레네(생장→론세스바예스, +780m), 폰세바돈(+560m), 오 세브레이로(+790m)

## 작업 방식

1. 작업 전 관련 파일을 먼저 읽는다. 추측으로 수정하지 않는다
2. 한 번에 하나의 관심사만 변경한다
3. 계산 로직을 건드리면 `npm test`를 돌리고 결과를 보고한다
4. 새 의존성을 추가하기 전에 이유를 먼저 설명한다
5. 커밋 메시지는 한국어, 명령형 (`일정 분할 엔진에 오르막 보정 추가`)
6. 확실하지 않으면 지어내지 말고 질문한다
