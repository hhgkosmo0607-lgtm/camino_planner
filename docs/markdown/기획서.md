# 서비스 기획서

> 한국인 순례자를 위한 카미노 데 산티아고 동반 서비스
> 2026년 7월

---

## 1. 개요

### 1.1 한 문장

> **순례자가 다치지 않고, 숙소 걱정 없이 걷게 한다.**

### 1.2 서비스명

**미확정.** 아래 기준으로 검토 중이다.

짧고 발음이 쉬울 것 · 검색 가능할 것 · 카미노 고유일 것 · 종교색이 짙지 않을 것.

후보: **세요**(Sello, 도장) · **모혼**(Mojón, 이정표) · **콘차**(Concha, 조가비)

이 문서에서는 '서비스'로 표기한다.

### 1.3 산티아고 순례길이란

#### 한 문장으로

스페인 북서부 도시 **산티아고 데 콤포스텔라**의 대성당까지, 한 달 남짓 매일 20~30km씩 걸어서 가는 길이다.

사도 야고보의 무덤이 발견됐다는 전승에서 9세기에 시작돼 중세 3대 성지가 됐고, 1993년 유네스코 세계유산에 등재됐다. **지금은 종교와 무관하게 걷는 사람이 대부분이며, 실질적으로는 세계에서 인프라가 가장 잘 갖춰진 장거리 도보 여행 코스다.**

**2025년 530,987명이 완주 증서를 받았다.** 역대 최고치이며 10년 전보다 90% 늘었다.

#### 관광이 아니라 걷기가 목적이다

```
아침 출발 → 20~30km 걷기 → 숙소 도착 → 휴식 → 다음 날 다시
```

이것이 30~40일 반복된다. 매일 다른 마을에서 자고 매일 다른 사람을 만난다.

#### 길은 여러 갈래이고, 우리는 하나만 다룬다

| 루트 | 2025년 | 비중 |
|---|---|---|
| **프랑스 길 (Camino Francés)** | **242,179명** | **46%** |
| 포르투갈 길 (중앙·해안) | 190,339명 | 36% |
| 그 외 (영국·프리미티보·북쪽 등) | 약 98,000명 | 18% |

**우리는 프랑스 길만 다룬다.** 한국인 대부분이 이 길을 걷고, 데이터 구축 비용이 루트마다 새로 들기 때문이다.

#### 프랑스 길 기본 정보

| 항목 | 값 |
|---|---|
| 출발 | 생장피드포르 (프랑스, 피레네 기슭) |
| 도착 | 산티아고 데 콤포스텔라 대성당 |
| 거리 | **약 773km** |
| 소요 | **30~35일** |
| 지형 | 피레네 산맥 → 메세타 고원 → 갈리시아 산지 |

#### 알아야 할 용어

| 용어 | 뜻 |
|---|---|
| **알베르게** | 순례자 전용 숙소. 다인실. 공립 8~12유로, 사립 12~20유로 |
| **크레덴시알** | 순례자 여권. 알베르게 입장권이자 완주 증명용 도장 수첩 |
| **세요** | 크레덴시알에 찍는 도장 |
| **콤포스텔라** | 완주 증서. 도보 100km(자전거 200km) 이상이어야 발급 |
| **노란 화살표** | 전 구간에 칠해진 길 표식. 길 찾기가 어렵지 않은 이유 |

#### 사리아가 최다 출발지인 이유

콤포스텔라 최소 요건이 **도보 100km**이고, 사리아가 그 조건을 만족하는 가장 가까운 큰 마을이다. 그 결과 **2025년 전체 순례자의 30.5%(162,076명)가 여기서 출발했다.**

#### 2027년 — 성년(Año Santo)

7월 25일 성 야고보 축일이 일요일인 해를 성년이라 하며, 순례자가 크게 는다. **직전 성년인 2010년은 전년 대비 85% 증가**했다(147,254 → 272,385명).

**2027년이 다음 성년이고, 그다음은 2032년이다.** 이것이 우리 프로젝트의 출시 시점을 결정한다.

## 2. 문제

### 2.1 핵심

> **순례자는 길을 못 찾아서 힘든 게 아니라, 혼자 판단할 근거가 없어서 힘들다.**

노란 화살표와 조가비 표식이 잘 되어 있어 길은 거의 잃지 않는다. 숙소와 식당도 곳곳에 있다. **생활은 어렵지 않다.**

어려운 것은 판단이다.

| 판단해야 하는 것 | 근거가 없는 이유 |
|---|---|
| 오늘 몇 km를 걸어야 무리가 없나 | 내 체력과 구간 난이도를 연결해주는 도구가 없다 |
| 어느 날 숙소를 예약하고 어느 날 비워둘까 | 시기·구간별 혼잡도를 알 수 없다 |
| 이 통증은 쉬면 낫나, 병원에 가야 하나 | 판단 기준이 없고 언어도 안 통한다 |
| 짐을 부치는 게 나은가 | 비용과 부상 위험의 트레이드오프를 아무도 계산해주지 않는다 |
| 신발을 새로 사야 하나 | 기준을 모른다 |

### 2.2 두 축

문제는 두 갈래이고, 서로가 서로의 원인이다.

| 축 | 문제 |
|---|---|
| **A. 몸** | 부상이 완주를 막는다 |
| **B. 잠자리** | 예약과 자유 사이에서 판단할 근거가 없다 |

```
  예약을 안 했다              예약을 다 했다
       ↓                          ↓
  자리가 걱정된다            오늘 25km를 가야만 한다
       ↓                          ↓
  새벽에 서둘러 걷는다        무릎이 아파도 걷는다
       ↓                          ↓
      부상  ←──────────────────→  부상
```

**판단이 없으면 어느 쪽으로 가도 부상에 이른다. 이 순환을 끊는 것이 제품이다.**

### 2.3 부상의 실체

발과 무릎이 가장 흔한 부상 부위다. 발은 온갖 지형과 거의 직접 닿고, 무릎은 체중 전체를 받는다.

주요 원인은 **과사용과 피로, 잘못된 장비(안 맞는 신발, 무거운 배낭), 훈련 부족, 발 관리 소홀, 잘못된 보행**으로 보고된다.

**다섯 중 넷이 출국 전에 결정된다.** 준비 단계가 중요한 이유다.

### 2.4 숙소의 실체

| 유형 | 예약 | 요금 (2026) |
|---|---|---|
| 공립·Xunta·본당·기부제 | **불가** (선착순, 1박 제한) | 8~12유로 |
| 사립 | **가능** (전화·WhatsApp·플랫폼) | 12~20유로 |

돈을 조금 더 내면 전 구간 예약이 가능하다. 예약 인프라 자체는 좋아졌다.

**그래서 문제는 "확보"가 아니라 "판단"이다.**

- 전부 예약하면 비싸고, 다치거나 컨디션이 좋을 때 바꿀 수 없다
- 아무것도 안 하면 성수기 사리아 이후에서 곤란해진다
- **그 사이에서 어느 날을 잡을지 판단할 근거가 없다**

## 6. 기능

### 6.0 프랑스 길은 하나의 선이 아니다

**11곳에서 갈라졌다 다시 만난다.** 어느 쪽을 걷느냐에 따라 거리·난이도·볼거리가 달라진다.

| 분기점 | 선택지 |
|---|---|
| **생장피드포르** | **나폴레옹(고지)** vs **발카를로스(계곡)** — 11/1~3/31 나폴레옹 폐쇄 |
| 레온 이후 | 비야당고스(차도) vs **비야르 데 마사리페** |
| 트리아카스텔라 | 산 실(짧음) vs **사모스(대수도원)** |
| 그 외 8곳 | |

출발일·체력·관심(경치/역사/한적함/최단)·차도 회피 여부로 **추천하고 이유를 함께 보여준다.**
계절 폐쇄 구간은 아예 선택지에서 뺀다.

⚠️ **당일 안전 판단은 현지 순례자 사무소가 한다.** 우리는 사전 정보만 제공한다.

### 6.0-b 걷기 전과 후도 여정이다

**생장피드포르에는 공항이 없다.** 인천에서 가려면 최소 2~3회 환승이 필요하고, 이것이 준비 단계에서 가장 헷갈리는 부분 중 하나다.

| 경로 | 구간 |
|---|---|
| 파리 경유 | 인천 → 파리(CDG) → TGV 바욘 → 생장 기차 |
| 비아리츠 경유 | 인천 → 파리 → 비아리츠 → 버스 → 바욘 → 생장 |
| **팜플로나 경유** | 인천 → 마드리드 → 팜플로나 → 생장 버스 |

일정에 **Day 0(이동)** 으로 표시하고, 귀환편까지 함께 계획한다.

**여정 중 이동수단도 예외가 아니다.** 부상·만실·시간 부족으로 버스나 택시를 쓰는 일은 흔하다.
걷지 않은 구간은 자동으로 기록에서 빠지고, **콤포스텔라 요건에 영향을 주면 경고한다.**

#### 모두가 걷는 것은 아니고, 걷는 사람도 계속 걷지는 않는다

| 방식 | 콤포스텔라 최소 | 하루 이동 |
|---|---|---|
| 도보 | 100km | **3~32km** |
| 자전거 | **200km** | 40~100km |
| **전기자전거** | **대상 아님** | — |
| 전동 핸드바이크 | 확인 필요 | 평균 20km |
| 휠체어·조엘레트 | 확인 필요 | 3~15km |
| 말 | 100km | — |

**하루 목표 거리에 하한을 두지 않는다.** 의족을 쓰고 하루 3~4km씩 걷는 계획이 실재한다.

##### ⚠️ 메세타 건너뛰기는 실패가 아니다

**부르고스에서 레온까지 약 200km의 메세타를 버스로 건너뛰는 것은 흔한 선택이다.** 포럼에서 끊임없이 논의되고 상당수가 실제로 그렇게 한다.

일정 수립 단계에서 **처음부터 넣을 수 있게** 한다. 걷지 않은 구간은 조용히 기록에서 빠지고, 콤포스텔라 요건에 걸릴 때만 경고한다.

**어느 쪽도 권하지 않는다.** 시간과 체력의 현실을 보여주고 판단은 사용자가 한다.

Phase 1~3은 도보를 우선하되, **데이터 모델과 용어를 도보 전용으로 만들지 않는다.** 나중에 넣으려면 전부 뜯어야 한다.

| **예산 관리** | 계획 예산과 실제 지출 대조. 오프라인 입력 |
| 체크리스트 | 준비·배낭·매일 아침·매일 저녁·완주 후 |
| **인쇄용 일정표** | A4 한 장. 자녀가 부모에게 뽑아준다 |

### 6.2 길 위에서

### 6.3 보여주기 카드

**화면을 상대에게 내밀면 대화 없이 상황이 끝난다.**

번역기와 다른 점은 두 가지다.

1. **무엇을 말해야 하는지 우리가 안다.** "물집이 생겼어요"를 번역해도 소용없다. 약사가 "며칠 됐는지"와 "터졌는지"를 알아야 판단하는데, 순례자는 그걸 모르니 말하지 않는다.
2. **상대의 답을 알아들을 수 있다.** 카드에 응답 선택지가 있어 상대가 짚으면 한국어로 표시된다.

| 분류 | 카드 |
|---|---|
| 숙소 (9장) | 침대 있나요 · 만실인데 어디로 · 짐배송 접수 · 건조기 · 빈대 신고 등 |
| 몸 (8장) | 약국(물집·근육통·관절) · 진료 필요 · 응급 · **내 의료 정보** · 보험 서류 |
| 이동·기타 (6장) | 택시 · 버스 · 길을 잃음 · 물 · 식사 제한 |

의료 카드는 **증상을 전달하고 전문가의 답을 받아오는** 도구다. 약을 지정하거나 진단하지 않는다.

배터리가 나가는 상황을 대비해 **핵심 8장은 인쇄본**으로도 제공한다.

### 6.4 완주 이후

**보상이 소유욕을 만들어야 한다.** 파일 하나로는 800km에 값하지 않는다.

| 동인 | 우리 장치 |
|---|---|
| **노력의 증거** | 걸어야만 열리는 지도 |
| **수집 완성** | 세요(도장) 아카이브, 랜드마크 19곳 |
| **개인화** | 내 경로, 내 날짜, 내 사진 |
| **물성** | 포스터, 포토북, 필름 사진 |
| **자랑 가능성** | 숫자 카드 |
| **애착** | 크레덴시알, 「내 돌」 |

#### 안개 지도

지도가 처음에는 구름에 덮여 있고, **걸으면 걸은 만큼 열린다.** 카미노의 상징적 장소 19곳에 도착하면 그 주변이 밝아지며 그곳의 이야기가 함께 열린다.

용서의 언덕, 이라체 와인 샘, 산토 도밍고 대성당, 철의 십자가, 오 세브레이로, 몬테 도 고소 등이다.

완주 후 **"내가 밝힌 지도"가 포스터로 남는다.** 디지털 성취가 벽에 걸리는 물건이 된다.

> **안개는 이야기 레이어만 가린다. 경로와 화살표, 알베르게는 항상 보인다.**
> 기본값은 꺼짐이고 언제든 끌 수 있다. 안전이 재미보다 위다.

#### 「내 돌」

순례자는 집에서 돌 하나를 가져와 **철의 십자가(1,505m)** 에 놓는다. 각자 그 돌에 무엇을 실을지 스스로 정한다. 프랑스 길 최고점이자 가장 상징적인 순간이다.

출발 전에 돌 사진과 한 줄을 등록해두면, 철의 십자가에 도착했을 때 그 문장이 다시 보인다. 완주 후에는 순례기와 지도 포스터에 남는다.

**마스코트를 만들지 않는다.** 이미 있는 의식을 기록해줄 뿐이다. 그래서 부담이 없고, 그래서 진짜다.

⚠️ 종교적 해석이나 위로의 문구를 덧붙이지 않는다. 무엇을 내려놓을지는 사용자가 정한다.

#### 다음 여정

카미노와 일본 구마노고도는 **세계에서 단 둘뿐인 유네스코 세계유산 순례길**이고, 양쪽을 걸으면 **이중 순례자(Dual Pilgrim)** 자격을 받는다. 조가비와 세 발 까마귀를 결합한 핀 배지와 화지 인증서를 준다.

**카미노를 먼저 걸은 사람이 유리하다.** 구마노 혼구 대사에서 다이코 의식에 참여할 수 있다. 항공 20~30만 원에 3~5일이면 되고, 최소 코스는 약 7km다.

한국 커뮤니티에 정리된 자료가 거의 없어, 안내 자체가 가치가 된다.
