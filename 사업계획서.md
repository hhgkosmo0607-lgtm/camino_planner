### 2.4 2027년 성년

**직전 성년인 2010년은 전년 대비 85% 증가했다.** 2025년 53만 명 기준을 적용하면 2027년은 70만 명 이상, 한국인은 1만 2천~1만 5천 명이 예상된다.

**다음 성년은 2032년이다.**

## 3. 경쟁

### 3.1 진짜 경쟁자는 앱이 아니다

```
       비용
        ↑
  600만 ┤                        ● 여행사 풀패키지
  400만 ┤                  ● 세미패키지
        │       ★ 우리 (20~50만)
   50만 ┤
    0원 ┤ ● 혼자 (카페 + 무료앱)
        └───────────────────────────→ 안심
```

| 대체재 | 무엇을 해결하나 | 가격 |
|---|---|---|
| **여행사 패키지** | **부상 + 숙소 전부** | 350~600만 |
| 네이버 카페 | 정보와 경험 | 무료 |
| 기존 앱 (Wise Pilgrim 등) | 길찾기, 숙소 목록 | 무료~$4.99 |
| 아무것도 안 함 | — | 0 |

### 3.2 기존 도구의 실체

순례자는 종이·웹·앱을 섞어 쓴다.

| 도구 | 특징 |
|---|---|
| **John Brierley 가이드북** | 2003년 초판. 가장 많이 들고 다니는 책 |
| **Wise Pilgrim 앱** | $4.99. 2026년 리뉴얼. 오프라인 지도, 예약 도우미 |
| **Buen Camino 앱** | 무료. 9개 언어. 루트 수 최다 |
| **Gronze / Godesalco (웹)** | 숙소 DB, 구간 계산기. 스페인어 |
| 생장 사무소 무료 인쇄물 | 거리·숙소·고도가 A4 한 장에 |

**이들이 이미 잘하는 것**: 길찾기, 숙소 목록, 고도 프로파일, 구간 계산

**아무도 하지 않는 것**

| 공백 | 설명 |
|---|---|
| **한국어** | 전부 지원하지 않는다 |
| **부상** | **어느 도구도 다루지 않는다** |
| **판단** | 데이터를 보여줄 뿐 결론을 내려주지 않는다 |
| **준비 단계** | D-90부터를 다루는 도구가 없다 |
| **현지 소통** | 보여주기 카드 같은 도구가 없다 |
| **갈림길 판단** | 11곳의 선택을 체력·계절·관심으로 추천하는 곳이 없다 |
| **계획 이탈 대응** | 만실·부상 시 대안을 만들어주는 곳이 없다 |

한국인은 전체 순례자의 **1.4%**다. 글로벌 도구가 한국어에 투자할 이유가 없다.

### 3.3 여행사가 파는 것

| | 자유여행 | 패키지 |
|---|---|---|
| 총비용 | 약 430~490만 | 약 500~600만 |
| **차액** | | **70~150만** |
| 어디서 자나 | **알베르게** | **호텔·펜시온** |

**차액은 100만 원 남짓이다. 그런데도 대부분은 자유여행을 간다. 알베르게에서 자고 싶어서다.**

즉 **패키지 고객은 돈이 많아서가 아니라 불안해서 산다.** 그리고 그 100만 원에 순례 경험의 핵심인 알베르게를 잃는다.

## 4. 사업 모델

### 4.1 두 축과 그 순환

```
  예약을 안 했다              예약을 다 했다
       ↓                          ↓
  자리가 걱정된다            오늘 25km를 가야만 한다
       ↓                          ↓
  새벽에 서둘러 걷는다        무릎이 아파도 걷는다
       ↓                          ↓
      부상  ←──────────────────→  부상
```

**이 순환을 끊는 지점이 수익 지점이다.**

| 축 | 역할 | 수단 |
|---|---|---|
| **B. 숙소** | **유입을 만든다** | 예약 판단, 혼잡 추정, 대안 제시 |
| **A. 부상** | **수익을 만든다** | 보험, 신발·장비, 짐배송 |

순례자는 신발을 사러 오지 않는다. **"숙소 어떡하지"로 온다.** B가 데려오고 A가 돈이 된다.

**그리고 A축은 대체 불가능하다.** 예약 플랫폼도, 패키지도, 번역기도 부상을 막아주지 않는다.

### 4.2 수익원

| # | 항목 | 축 | 인당 |
|---|---|---|---|
| R1 | **여행자 보험 제휴** | A | 2~5만 |
| R2 | **신발·장비 커머스** | A | 3~8만 |
| R3 | 숙소 예약 제휴 | B | 1~3만 |
| R4 | 짐 배송 연계 | A+B | 1~2만 |
| R5 | **반패키지 상품** | A+B | 20~50만 |
| R6 | 여행사 송객 | — | 20~40만 |
| R7 | 완주 실물 세트 | — | 3~8만 |
| R8 | B2B 단체 순례 | A+B | 건당 50~100만 |

핵심 기능은 **무료**다. 유료화는 도달률의 상한을 만들기 때문이다. 수익은 사용자가 어차피 쓸 돈에서 나온다.

### 4.2-b 제품이 실제로 하는 일

| 시점 | 기능 |
|---|---|
| 준비 | 부상 회피 일정 설계 · **갈림길 11곳 추천** · 예약할 날 지정 · 비용·배낭 계산 · 인쇄용 일정표 |
| 길 위 | 오늘 구간 · "서두를 필요 있나요?" · 오프라인 지도 · **보여주기 카드 23장** · **Plan B** · 예산 기록 |
| 완주 후 | **안개 지도** · 세요 아카이브 · 숫자 카드 · 이중 순례자 안내 · 실물 상품 |

#### 프랑스 길은 하나의 선이 아니다

**11곳에서 갈라졌다 다시 만난다.** 어느 쪽을 걷느냐에 따라 거리·난이도·볼거리가 달라진다.

대표적으로 생장피드포르에서 **나폴레옹(고지)** 과 **발카를로스(계곡)** 로 갈리는데, **나폴레옹은 11월 1일부터 3월 31일까지 폐쇄된다.** 오리송 위로 대피할 곳이 없고 날씨가 몇 분 만에 바뀌기 때문이다.

**어느 앱도 이 선택을 도와주지 않는다.** 우리는 출발일·체력·관심으로 추천하고 이유를 함께 보여준다.

#### 순례는 인천공항에서 시작한다

**생장피드포르에 공항이 없다.** 최소 2~3회 환승이 필요하고, 이것이 준비 단계에서 가장 헷갈리는 부분이다. 파리·비아리츠·팜플로나 경유 세 가지 주요 경로를 비교해 보여주고 **일정의 Day 0으로 넣는다.**

**여정 중 이동수단도 예외가 아니다.** 부상·만실·시간 부족으로 버스나 택시를 쓰는 일은 흔하다. 걷지 않은 구간은 기록에서 빠지고, **콤포스텔라 요건(도보 100km 연속)에 영향을 주면 경고한다.**

#### 계획은 반드시 어긋난다

숙소가 만실이거나 몸이 아프면 **대안 4안(더 간다 / 되돌아간다 / 이동수단 / 사립 예약)** 을 만들고 남은 일정을 다시 나눈다.

**별도 화면이 아니라 각 날짜 카드 안에 접혀 있다.** 그날 정보를 보면서 판단해야 하기 때문이다.

⚠️ 콤포스텔라 요건은 **2025년에 개정됐다.** 도보·말 100km, 자전거 200km, 전기자전거는 대상이 아니다. 연속이어야 하고 마지막 구간이 최소 20km여야 한다.

### 4.3 반패키지 — 최대 기회

**패키지 고객 1명이 자유여행 고객 10~15명의 가치를 갖는다.**

| 구분 | 우리 매출 |
|---|---|
| 자유여행 사용자 | 2~3만 |
| **패키지 전환자** | **20~50만** |

#### 전환 가능 세그먼트

| 유형 | 비중 | 전환 |
|---|---|---|
| **불안해서** | 30% | ✅ |
| **시간이 없어서** | 20% | ✅ |
| **가족이 걱정해서** | 15% | ✅ |
| 언어가 두려워서 | 15% | 🟡 |
| 혼자가 외로워서 | 10% | ❌ |
| 진짜 편하고 싶어서 | 10% | ❌ |

**65%가 전환 가능 시장이다.**

#### 상품 두 단계

| 상품 | 가격 | 내용 |
|---|---|---|
| **안심 준비** | 15~25만 | 일정 설계, **갈림길 선택**, **일자별 상세**, 예약할 날 지정, 예산 계획. **실행은 본인** |
| **안심 동행** | 40~60만 | 위 전부 + **실제 예약 대행**(제휴 여행사 실행) + 한국어 비상 지원 |

#### 가족 안심 세트

**60대 부모를 보내는 자녀가 결제자인 경우가 많다.**

매일 도착 알림, 예정 시각 초과 시 알림, 3일 미기록 시 확인 요청, 가족용 웹페이지, 비상 연락 카드.

**원가가 거의 0이고 여행사도 제대로 팔지 않는다. 자녀가 부모에게 사주는 상품이 된다.**

⚠️ 실시간 위치 공유는 하지 않는다. "도착했다"는 사실만 순례자가 직접 전달한다.

### 4.4 여행사와의 관계

**경쟁이 아니라 파이를 넓히는 구조다.**

```
현재      자유여행(80%) ─────────── 패키지(20%)
              ↑ 여행사 매출 0

이후      자유여행(65%) ─ 반패키지(15%) ─ 패키지(20%)
                            ↑ 여행사가 실행 → 신규 매출
```

반패키지는 여행사에 **없던 상품 라인**이다. 마케팅 비용 없이 고객을 받는다.
그리고 **패키지가 맞는 사용자는 우리가 여행사로 안내한다.**

## 9. 확장

**부상과 숙소는 카미노만의 문제가 아니다. 모든 장거리 도보의 문제다.**

| 시장 | 규모 | 연결 |
|---|---|---|
| **구마노고도** | — | **이중 순례자 제도.** 완주자가 자연스럽게 다음 목표로 삼는다 |
| 제주올레 (437km) | 연 90만+ | 구간 설계 엔진 그대로 |
| 해파랑길 (750km) | 수십만 | 숙소 문제가 더 크다 |
| 시코쿠 88 순례길 | 성장세 | 순례 + 도장 문화 동일 |

카미노는 매출 5~6억 규모지만, **한국 도보여행 카테고리로 나가는 교두보다.**
