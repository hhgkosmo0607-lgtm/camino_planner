# 1부 · 경쟁과 대체재

## 1. 경쟁 지형을 다시 그린다

이전 문서는 앱끼리 비교했다. **틀린 프레임이다.**

한국인 순례자는 "카미노 앱을 고르는" 게 아니라 **"이 여행을 어떻게 감당할지"** 를 고른다. 그 선택지가 진짜 경쟁 지형이다.

```
       비용
        ↑
  600만 ┤                        ● 여행사 풀패키지
        │
  400만 ┤                  ● 세미패키지
        │
        │       ★ 우리 (20~50만)
   50만 ┤
        │
    0원 ┤ ● 혼자 (카페 + 무료앱)
        └───────────────────────────→ 안심
          불안                    안심
```

**우리는 "혼자"와 "세미패키지" 사이의 빈 곳을 노린다.**

## 2. 대체재별 심층 분석

### 2.1 여행사 패키지 — 가장 중요한 경쟁자

#### 상품 구조

| 유형 | 가격대 | 구성 |
|---|---|---|
| 풀패키지 (인솔자 동행) | 500~600만 | 항공 + 전 구간 숙소 + 짐배송 + 인솔 + 일부 식사 |
| 세미패키지 | 350~450만 | 항공 + 숙소 + 짐배송, 인솔 없음 |
| 현지 지원형 | 200~300만 | 숙소 + 짐배송만 |

실제 국내 상품 사례에서 **전 구간 짐 배송이 약 200유로**로 별도 표기되고, 43일 일정이 구성된다.

#### 왜 사람들이 이걸 사는가

| 이유 | 우리 축 |
|---|---|
| 숙소를 못 잡을까 봐 | **B** |
| 언어가 안 통해서 | A + B |
| 나이가 있어서 몸이 걱정돼서 | **A** |
| 혼자 가기 무서워서 | — |
| 알아볼 시간이 없어서 | A + B |

**→ 다섯 중 넷이 우리 두 축이다.** 여행사는 우리와 같은 문제를 팔고 있다.

#### 약점

| 약점 | 우리 기회 |
|---|---|
| 비싸다 (10~30배) | 가격 |
| 일정이 고정 | 자유도 |
| 알베르게가 아니라 호텔 — **순례 경험이 희석된다** | **진정성** |
| 만나는 사람이 한국인 일행뿐 | 경험 |
| 예약이 전 구간이라 부담 | 부분 선택 |

> **핵심 공격 포인트: "패키지로 가면 알베르게에 안 묵는다."**
> 순례 경험의 대부분이 알베르게에서 일어난다. 여행사는 이걸 못 준다.
> **우리는 알베르게에 묵으면서 안심할 수 있게 한다.**

#### 그런데 적이 아니다

**여행사는 R5(반패키지)의 실행 파트너이자 R6(송객)의 대상이다.**

- 자유여행을 준비하다 포기하는 사람 → 여행사로 송객 (수수료)
- 반패키지 → 여행사가 실제 예약·결제 (화이트라벨)

**→ 인터뷰 때 "자유여행 준비하다 패키지로 돌아서는 분이 있나요? 왜요?"를 반드시 묻는다.** 이 답이 우리 상품의 설계도다.

### 2.2 네이버 카페 — 정보의 실질 허브

| 제공 | 강도 |
|---|---|
| 실시간 현지 정보 | ★★★ |
| 경험자의 답변 | ★★★ |
| 동행 구하기 | ★★★ |
| 정서적 지지 | ★★★ |
| 구조화된 정보 | ★ |
| 검색성 | ★ |

#### 이기는 곳 / 못 이기는 곳

| 질문 | 승자 |
|---|---|
| "사리아 5일 일정 짜줘" | **우리** (계산) |
| "포르토마린 알베르게 어때요?" | 카페 (경험) |
| "무릎 아픈데 어떡하죠?" | 카페 (공감) + 우리 (판단) |
| "지금 만실인가요?" | **카페** (실시간) |
| "신발 뭐 사요?" | 카페 (경험) → **우리가 노릴 곳** |
| "며칠 걸리나요?" | **우리** |

> **카페와 싸우지 않는다.** 카페는 경험과 실시간을, 우리는 계산과 판단을 맡는다.

#### 접근 원칙

| 하는 것 | 하지 않는 것 |
|---|---|
| 운영진에 사전 허락받고 무료 도구 공유 | 무단 홍보 글 |
| 질문에 성실히 답하기 (링크 없이) | 답변마다 링크 걸기 |
| 실측 도보 연재로 자연스러운 인지 | 바이럴 알바 |
| 인터뷰 모집도 허락받고 | 회원 DM |

⚠️ **네이버 카페는 상업적 접근에 매우 적대적이다. 한 번 찍히면 복구가 안 된다.**

### 2.3 글로벌 앱 — 우리 축과 다르다

| 앱 | 강점 | **두 축 대응** |
|---|---|---|
| Buen Camino | 45개 루트, 100만+ 다운로드, 9개 언어, 무료 | 숙소 **목록**. 부상 축 없음 |
| Wise Pilgrim | 14년 업력, 오프라인 지도, 2026 리뉴얼, $4.99 | 숙소 목록 + 예약 연결. 부상 축 없음 |

**→ 어느 앱도 "부상"을 다루지 않는다.** 목록은 주지만 판단은 안 준다.

**한국인은 전체 순례자의 1.4%**라 이들이 한국어에 투자할 이유가 없다.

### 2.4 "아무것도 안 함" — 가장 큰 경쟁자

**절반 이상은 카페 몇 번 보고 그냥 간다.** 이기는 방법은 하나뿐이다. **검색했을 때 우리가 거기 있는 것.**

# 3부 · 부상 콘텐츠 설계

## 0. 이 문서의 위치

부상(A축)이 **수익의 절반 이상**을 만든다. 그런데 그 근거가 되는 콘텐츠 45개는 아직 목차조차 없었다.

동시에 **가장 위험한 영역**이다. 잘못 쓰면 사람이 다치고 우리가 법적 책임을 진다.

**이 문서는 무엇을 쓸지와, 쓰기 전에 무엇을 지켜야 하는지를 정한다.**

---

## 1. ⚠️ 대원칙 — 먼저 읽는다

### 1.1 우리가 하는 것과 하지 않는 것

```
✅ 하는 것
   · 무엇이 흔히 보고되는지 알린다
   · 준비 단계에서 무엇을 결정해야 하는지 알린다
   · 일정을 무리하지 않게 설계한다
   · 전문가에게 갈 수 있게 돕는다 (카드, 위치, 언어)
   · 전문가의 판단을 일정에 반영한다

❌ 하지 않는 것
   · 진단한다 ("족저근막염입니다")
   · 약을 지정한다 ("이 약을 드세요")
   · 용량·복용법을 쓴다
   · "괜찮다"고 안심시킨다
   · 치료법을 처방한다
```

**우리는 통로다. 판단자가 아니다.**

### 1.2 표현 규칙

| ❌ | ✅ |
|---|---|
| "족저근막염입니다" | "발뒤꿈치 통증은 여러 원인이 있습니다" |
| "이 약을 드세요" | "약국에서 증상을 설명하는 카드를 준비했습니다" |
| "3일 쉬면 낫습니다" | "쉬는 기간은 전문가와 상의하세요" |
| "괜찮습니다" | "판단이 어려우면 진료를 받으세요" |
| "치료법은" | "일반적으로 알려진 대응은" |
| "반드시 ~하세요" | "~하는 것이 권장됩니다" |

### 1.3 모든 페이지 하단 고정

## 2. 의료 검수 프로세스

### 2.1 검수 없이 게시하지 않는다

**45개 페이지 전부가 검수 대상이다. 예외 없다.**

```
① 초안 작성          ← 우리
② 사실 확인          ← 공개 자료 대조
③ ★ 의료 자문 검수   ← 정형외과 또는 스포츠의학
④ 표현 수정          ← 1.2절 규칙 적용
⑤ 재검수            ← 수정본 확인
⑥ 게시 + 검수일 표시
⑦ 연 1회 재검토
```

### 2.2 자문 계약

| 항목 | 내용 |
|---|---|
| 대상 | 정형외과 전문의 또는 스포츠의학 전문의 |
| 보조 | 물리치료사 (보행·재활 영역) |
| 형태 | 월 정액 또는 건당 |
| 범위 | 콘텐츠 검수 + 보여주기 카드 의료 8장 + 위험 산정 로직 검토 |
| 예상 비용 | **300~500만** |
| 시점 | **부상 콘텐츠 착수 전 (2026년 11월)** |

### 2.3 검수 요청 양식

```
[검수 요청]
페이지: (제목)
목적: 순례 준비자에게 ○○을 알리기 위함
대상: 40~70대 일반인, 의학 지식 없음

확인 부탁드릴 것
□ 사실관계 오류
□ 진단·처방으로 읽힐 여지
□ 위험한 자가 대응을 유도하는 부분
□ "전문가에게 가야 하는 신호"가 빠졌는지
□ 표현 강도가 적절한지
```

### 2.4 ⚠️ 검수자가 거부하면 게시하지 않는다

**"이건 쓰지 마세요"라고 하면 안 쓴다.** 다시 설득하지 않는다.

```

**핵심은 "우리가 뭘 팔지"가 아니라 "이 시장이 있다"를 먼저 보여주는 것이다.**

### 4.2 접촉 대상

| 유형 | 장단 |
|---|---|
| 브랜드 본사 | 단가 좋음, 절차 느림, 품목 제한 |
| **아웃도어 유통** | **품목 다양, 빠름. 우선** |

**단순 링크가 아니라 "순례 특화 큐레이션"을 제안한다.**

```
■ 저희가 드릴 수 있는 것
- 출국 D-60 시점의 구매 수요 (신발 길들이기 기간)
- 개인별 일정 데이터 기반 추천

■ 요청
- 제휴 수수료
- 순례자 대상 기획전 또는 할인 코드

⚠️ **"할인 코드"가 수수료보다 사용자에게 가치가 클 수 있다.** 우리 신뢰도 올라간다.

## 5. 2027 성년 시즌 대응

### 5.1 왜 이 시즌이 전부인가

**2027년은 성년(Año Santo Xacobeo)이다.** 직전 성년 2010년은 전년 대비 85% 급증했다. **다음 성년은 2032년.**

### 5.2 순례 시즌 달력

| **9~10월** | **2차 성수기** | **최대 부하** |
| 11월 | 급감 | 시즌 회고 |

**"무릎이 아픈데 걸어도 되나요?" 에 답하지 않는다.**

- 진단·판단을 하지 않는다
- 현지 의료기관 정보와 보험 청구 절차만 안내
